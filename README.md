# Sheweny

Sheweny is a framework for create discord bots with javascript (or typescript).

[DOCUMENTATION](https://sheweny.js.org)

## Getting Started

Prerequisites

Node.js 16.6.0 or newer is required.

### Instalation

With npm :

```sh-session
npm install sheweny
```

With yarn :

```sh-session
yarn add sheweny
```
